# Sales Dashboard Analysis using Power BI

## Project Overview

This project is an interactive sales dashboard built using Power BI. The dashboard analyzes sales performance across different states, product categories, and time periods.

## Objectives

* Analyze sales performance by region and category
* Visualize sales trends over time
* Create interactive filters for better analysis
* Generate business insights from raw sales data

## Tools & Technologies Used

* Power BI
* Excel / CSV Dataset
* Data Visualization Techniques

## Dashboard Features

* Sales by State
* Sales by Category
* Sales Trends Over Time
* Interactive Filters (State & Category)
* KPI Cards for Total Sales and Average Sales

## Key Insights

* California generated the highest sales among all states
* Technology category showed strong sales performance
* Sales trends varied across different time periods

## Files Included

* Power BI Dashboard (.pbix)
* Dataset File
* Dashboard Screenshots

## Conclusion

This project demonstrates how Power BI can transform raw sales data into interactive visual dashboards for business decision-making and performance analysis.
